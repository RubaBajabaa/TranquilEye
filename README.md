# 🧠 TranquilEye
An AI-based browser extension that detects overstimulating YouTube videos for children and redirects them to safer alternatives.
--- ## 🔍 Overview 
TranquilEye is designed to help parents protect children from overstimulating digital content.
The system analyzes video characteristics such as motion intensity, colors, and sound patterns to identify potentially overwhelming content and guide users toward safer viewing options. 
---## 🎯 Features 
* Detects overstimulating YouTube videos
* Suggests safer alternative content
* Provides a dashboard for parents to monitor viewing behavior
* Enhances safer digital experiences for children ---
