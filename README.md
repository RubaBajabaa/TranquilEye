## 🧠 TranquilEye

An AI-based browser extension designed to detect overstimulating YouTube content for children and promote safer viewing experiences.

---

## 🔍 Overview

TranquilEye analyzes video elements such as motion intensity, color patterns, and audio levels using machine learning techniques. Based on this analysis, it identifies potentially overstimulating content and helps redirect children to safer alternatives, supporting parents in managing digital exposure.

---

## 🎯 Key Features

- Detects overstimulating YouTube videos using AI
- Suggests safer alternative content
- Provides insights into children's viewing behavior
- Supports a healthier and safer digital environment
